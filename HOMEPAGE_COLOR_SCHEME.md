# HomePage Color Scheme - Beautiful & Professional

## 🎨 Color Palette

### Primary Colors
- **Blue**: `#2563EB` (Primary brand color)
- **Purple**: `#7C3AED` (Secondary accent)
- **Pink**: `#DB2777` (Tertiary accent)
- **Green**: `#059669` (Success/positive actions)

### Background Colors
- **Light Gray**: `#F8FAFC` (Main background)
- **White**: `#FFFFFF` (Card backgrounds)
- **Gradient Backgrounds**:
  - Hero: `linear-gradient(135deg, #EFF6FF 0%, #F5F3FF 50%, #FDF4FF 100%)`
  - Features: `linear-gradient(180deg, #FFFFFF 0%, #F8FAFC 100%)`
  - CTA: `linear-gradient(135deg, #1E293B 0%, #334155 100%)`

### Text Colors
- **Primary Text**: `#1E293B` (Dark slate)
- **Secondary Text**: `#64748B` (Medium gray)
- **Light Text**: `#CBD5E1` (Light gray for dark backgrounds)

### Accent Colors (for features)
- **Blue**: `#2563EB` - AI-Powered Planning
- **Purple**: `#7C3AED` - Vendor Marketplace
- **Pink**: `#DB2777` - Guest Management
- **Green**: `#059669` - Budget Tracking
- **Red**: `#DC2626` - Timeline Planning
- **Orange**: `#EA580C` - Team Collaboration

## 📐 Section Breakdown

### 1. Hero Section
**Background**: Soft gradient from blue to purple to pink
- `linear-gradient(135deg, #EFF6FF 0%, #F5F3FF 50%, #FDF4FF 100%)`
- Subtle radial gradients for depth
- Badge: Blue to purple gradient with white text
- Heading: Blue to purple to pink gradient text
- Subtext: Medium gray (`#475569`)
- Primary Button: Blue gradient (`#3B82F6` to `#2563EB`)
- Secondary Button: Blue outline

### 2. Stats Section
**Background**: Pure white (`#FFFFFF`)
- Each stat card has its own color:
  - Events: Blue (`#2563EB`)
  - Users: Purple (`#7C3AED`)
  - Vendors: Pink (`#DB2777`)
  - Success: Green (`#059669`)
- Cards have gradient backgrounds with their respective colors
- Hover effect: Lift with colored shadow

### 3. Features Section
**Background**: White to light gray gradient
- 6 feature cards, each with unique color:
  - AI Planning: Blue (`#2563EB`)
  - Vendor: Purple (`#7C3AED`)
  - Guest: Pink (`#DB2777`)
  - Budget: Green (`#059669`)
  - Timeline: Red (`#DC2626`)
  - Team: Orange (`#EA580C`)
- Icon containers with colored backgrounds
- Hover: Icon background becomes gradient, card lifts

### 4. Benefits Section
**Background**: Pure white (`#FFFFFF`)
- 4 benefit cards with circular icon containers
- Colors:
  - Easy to Use: Blue (`#2563EB`)
  - All-in-One: Purple (`#7C3AED`)
  - Verified: Pink (`#DB2777`)
  - Secure: Green (`#059669`)
- Gradient backgrounds on icon circles
- Hover: Border color intensifies

### 5. CTA Section
**Background**: Dark gradient (`#1E293B` to `#334155`)
- White text for contrast
- Light gray subtext (`#CBD5E1`)
- Blue gradient button
- Radial gradient overlay for depth
- Green checkmarks for trust indicators

### 6. Footer
**Background**: Light gray (`#F8FAFC`)
- Gray text (`#64748B`)
- Blue hover color on links (`#2563EB`)

## ✨ Visual Effects

### Hover Effects
- **Cards**: Lift up 8px with colored shadow
- **Buttons**: Darker gradient + stronger shadow
- **Icons**: Background becomes gradient, color changes to white
- **Links**: Color changes to blue

### Shadows
```css
Button: 0 4px 14px rgba(37,99,235,0.4)
Button Hover: 0 6px 20px rgba(37,99,235,0.5)
Card Hover: 0 12px 24px [color]25
```

### Transitions
- All transitions: `0.3s ease`
- Smooth and professional feel

## ♿ Accessibility

### Contrast Ratios
- **Primary Text on White**: 14.5:1 (AAA)
- **Secondary Text on White**: 7.5:1 (AAA)
- **White Text on Blue**: 8.6:1 (AAA)
- **White Text on Dark**: 15.8:1 (AAA)

### Color Blindness
- Multiple visual cues beyond color
- Icons accompany colored elements
- Text labels for all features
- High contrast maintained
